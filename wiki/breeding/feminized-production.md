# Feminized Seed Production Endpoint

The long-term objective is not just "make seeds." The objective is a repeatable system that can produce large feminized seed batches where a high percentage of offspring reliably express the target:

- dark purple buds, stems, and leaves
- vigorous sativa-leaning / trellis-friendly structure
- acceptable terpene appeal, flower quality, vigor, finish time, and resilience
- no hermaphrodite tendency

Working target: a production seed lot should not be treated as stable until progeny testing shows **>90% on-target females** under the normal grow environment.

## When this phase starts

Do not use feminized production as the main early shortcut. Use regular male pollen for the first F2 and early F-generation work because males expose and recombine variation quickly.

Feminized production becomes the main method only after a family has already performed well in progeny testing, likely around F4/F5 unless an earlier family strongly exceeds expectations.

## Clone roles

The final production system maintains elite female clones, not a permanent male plant.

| Role | What it is | Why it exists |
|---|---|---|
| Seed mother | Elite female clone from the best progeny-tested family | Carries the flower and seed crop |
| Reversal donor | Elite female clone from the same proven family, or the same clone if selfing is intentional | Produces feminized pollen after chemical reversal |
| Anchor clone | A/D or another historic elite clone | Backup reference if a line drifts away from the target |
| Backup stasis clones | Small copies of candidate females | Insurance while test results mature |

There is no long-term "father plant" in the final feminized system. The pollen donor is a female clone that is reversed when needed. True males remain useful for earlier F-generation work and for preserving genetic range, but they are not the final feminized production parent.

## Production workflow

1. Select the best family from a completed progeny test.
2. Keep 1-2 elite female clones from that family in stasis or productive-mother mode.
3. Choose one clone as the seed mother and one clone as the reversal donor.
4. Reverse the donor with STS or another controlled reversal method in isolation. Do not consume any chemically treated reversal plant.
5. Collect, dry, label, and aliquot feminized pollen separately.
6. Pollinate the seed mother in isolation. For validation lots, branch pollination is enough; for production lots, whole-plant pollination can be used once the line is trusted.
7. Harvest, dry, count, and label the feminized seed lot.
8. Run a progeny test before calling the lot stable or using it as the new production baseline.

## Validation threshold

A seed lot advances only if the test population passes the hard gates from [stabilization-strategy.md](stabilization-strategy.md):

- early purple stem/petiole expression
- sativa-leaning, trellis-friendly structure
- dark bud purple by mid flower
- no hermaphrodite tendency
- acceptable vigor and flower quality
- known labels and clean parentage

Record the denominator. "Most looked right" is not enough. The project log should track:

- seeds germinated
- seedlings culled before flip, with reasons
- females flowered
- on-target females
- hermaphrodite/intersex failures
- off-target morphology/color failures
- final on-target percentage

Target interpretation:

| Result | Meaning |
|---|---|
| <70% on target | Not production-stable; return to family selection or backcross logic |
| 70-90% on target | Promising but still selection material |
| >90% on target | Candidate production lot; repeat once before relying on it as the stable line |

## Batch scale

Early validation lots should stay small: one or two pollinated branches usually produce enough seed for testing. Large batches come later, after the family repeatedly passes progeny testing.

For a trusted production run, flower a dedicated seed mother in isolation and pollinate more of the plant. The same labeling, contamination control, seed maturation, and storage rules from [cross-procedure.md](cross-procedure.md) still apply.

## Failure rules

- If the reversal donor throws intersex traits outside the intended reversal sites, discard the pollen and do not advance that clone.
- If the seed mother shows hermaphrodite tendency, discard the seed path.
- If labels are uncertain, discard the lot from breeding use.
- If a feminized lot produces off-type plants above the allowed threshold, keep the best individuals only as selection material; do not call the lot stable.
